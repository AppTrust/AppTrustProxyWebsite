# App Trust Proxy Website

Official Astro and Cloudflare website for App Trust Proxy.

App Trust Proxy Core is the free open source proxy runtime. App Trust Proxy is the desktop app for Mac that is still under development.

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Cloudflare check

```bash
npm run check
```

The site URL is configured as `https://apptrustproxy.com`.
